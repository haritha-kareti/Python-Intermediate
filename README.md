<h1> Python Intermediate Programming</h1>

<p>
This repository contains structured Jupyter Notebooks that cover 
<b style="color:#1e90ff;">Intermediate Python Programming Concepts</b>.<br>
It is designed for students who want to strengthen their programming logic 
before moving into advanced fields like Data Science, Machine Learning, 
and Web Development.
</p>

</div>

<hr>

<h2> Topics Covered</h2>
<ul>
    <li>Control Statements & Conditional Statements</li>
    <li>Functions</li>
    <li>Lambda Functions</li>
    <li>Comprehensions</li>
    <li>File Handling</li>
</ul>

<hr>

<h2>1️⃣ Control Statements & Conditional Logic</h2>
<p>Used to control the flow of execution in programs.</p>
<ul>
    <li>if, elif, else statements</li>
    <li>Nested conditions</li>
    <li>for loop</li>
    <li>while loop</li>
    <li>break, continue, pass</li>
    <li>Logical & comparison operators</li>
</ul>

<hr>

<h2>2️⃣ Functions</h2>
<ul>
    <li>Defining functions using def</li>
    <li>Parameters & arguments</li>
    <li>Default arguments</li>
    <li>Return statements</li>
    <li>Recursive functions</li>
    <li>Local & Global scope</li>
</ul>
<p><b>Functions improve code reusability and readability.</b></p>

<hr>

<h2>3️⃣ Lambda Functions</h2>
<ul>
    <li>Anonymous functions</li>
    <li>Lambda with multiple arguments</li>
    <li>Using with map(), filter(), reduce()</li>
</ul>
<p><b>Used for short-term, one-line function definitions.</b></p>

<hr>

<h2>4️⃣ Comprehensions</h2>
<ul>
    <li>List comprehensions</li>
    <li>Dictionary comprehensions</li>
    <li>Set comprehensions</li>
    <li>Conditional comprehensions</li>
    <li>Nested comprehensions</li>
</ul>
<p><b>Provide a clean and optimized way to create collections.</b></p>

<hr>

<h2>5️⃣ File Handling</h2>
<ul>
    <li>Opening files (r, w, a modes)</li>
    <li>Reading and writing files</li>
    <li>Appending data</li>
    <li>Using with statement</li>
</ul>
<p><b>Essential for real-world applications like data storage and logging.</b></p>
